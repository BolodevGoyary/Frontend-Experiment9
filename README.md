# Frontend-Experiment9: Basic Calculator

## Description
This project is a basic four-function calculator built using HTML, CSS, and JavaScript.  
It allows users to perform addition, subtraction, multiplication, and division.  
The interface includes buttons for numbers, operators, clear, delete, and equals.

## Features
- Addition, subtraction, multiplication, division
- Clear and delete functions
- Decimal support
- Responsive button layout

## How to Run
1. Clone the repository or download the files.
2. Open `index.html` in a web browser.
3. Click the buttons to input numbers and operators.
4. Press `=` to see the result.
